# 🏋️‍♂️ SportPro - Landing Page Deportiva

Landing page moderna, responsive y optimizada para conversión, enfocada en servicios de entrenamiento deportivo.

---

## 🌐 Demo

Puedes ver el proyecto en funcionamiento abriendo el archivo `index.html` en tu navegador.

---

## ✨ Características

* 🎯 Diseño tipo **landing page** (enfocado en conversión)
* 📱 Totalmente **responsive** (móvil, tablet y desktop)
* ⚡ Carga rápida y buen rendimiento
* 🎨 Estilo moderno con gradientes y efectos
* 🧭 Navegación simple e intuitiva
* 📩 Formulario de contacto funcional (simulado con JavaScript)
* 🔥 Secciones clave:

  * Hero principal
  * Beneficios
  * Call To Action (CTA)
  * Contacto

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (básico)

---

## 🎨 Personalización

Puedes modificar fácilmente:

* Colores → en el CSS (`background`, `color`)
* Textos → directamente en el HTML
* Imágenes → reemplazando la URL de fondo en `.hero`
* Botones → cambia los textos o acciones
